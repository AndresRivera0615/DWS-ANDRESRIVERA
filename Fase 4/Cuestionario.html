Cuestionario de Css Grid

1. Css Grid es un modelo de..

  1. Diseño css que permite crear paginas
  2. Gestionar imagenes por cuadriculas
  3. Poner mas bonita una pagina web
  4. Crear elementos de paginas web

2.CSS Grid divide la página en una cuadrícula o rejilla (grid) el cual usa tanto filas como columnas a la vez.

  1. Falso
  2. Verdadero

3. Cual es el primer atributo para trabajar con Css Grid

  1. Grid element en una etiqueta
  2. Display: grid en un contenedor
  3. justify-content en un container/li>
  4. Background GRID

4. Como funciona CSS Grid

  1. Dividir las secciones principales de una web en cuadrículas (filas y columnas)
  2. Ajusta imagenes en la posición que queramos
  3. Modificar una pagina web
  4. Cambiar el tamaño de los elementos de la web

5. Cual de las siguientes respuestas es una caracteristica de Css Grid

  1. Colocar los elementos en cualquier ubicación exacta de la cuadrícula
  2. Contralar la alineación, tanto de elementos, como de toda la cuadrícula
  3. Todas las anteriores
  4. Cambiar los elementos de la cuadrícula de sitio en cualquier momento de manera sencilla

6. Que es un area en Css Grid

  1. Es una o más grid cells (en-US) que forman un área rectangular en la cuadrícula
  2. Es el espacio que ocupa un elemento en la web
  3. Es la zona de la pantalla
  4. Son cuadros a al azar colocados en la web
